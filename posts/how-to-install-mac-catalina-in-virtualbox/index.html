How to Install macOS Catalina in VirtualBox (2021) | Step-by-Step Guide with Download Links

July 19, 2023 · 5 min · 864 words · Mary Jemmott
Table of Contents

  • Installing VirtualBox
  • Preparing for Installation
  • Running the VirtualBox Command Files
  • Installing macOS Catalina

Welcome back to the GEEKrar Guides channel. In this article, we will provide you with a detailed step-by-step guide on how to install macOS Catalina on VirtualBox on a Windows 10 PC. We understand that not every installation goes smoothly, and we have created a forum on our website where you can find solutions to common problems and get your questions answered. So, let’s get started with the installation process.

System Requirements#

Before we begin, let’s go over some system requirements. Here’s what you will need:

  • Windows 10 operating system
  • Minimum of 6GB RAM (8GB recommended)
  • At least 60GB of free hard disk space (80GB recommended)
  • VirtualBox and Extension Pack
  • macOS Catalina file
  • Command file for adjusting settings
  • Screen resolution file for full-screen capabilities

To make things easier for you, we have provided download links for all the necessary files in the description below. We also recommend checking the description for timestamps, so you can skip ahead if you have already installed VirtualBox.

Installing VirtualBox#

If you haven’t already installed VirtualBox, follow these steps:

  1. Open your browser and go to the VirtualBox website.
  2. Download the version of VirtualBox for Windows.
  3. Also, download the Extension Pack.
  4. Once downloaded, install VirtualBox by clicking ‘Next’ on the installation prompts and keeping all default settings.
  5. After the installation is complete, open VirtualBox Manager.
  6. Go to ‘Preferences’ and click on ‘Extensions’ in the left-hand menu.
  7. Click on the plus sign and add the Extension Pack you downloaded.
  8. Click ‘Install’, scroll down, and select ‘I Agree’ to complete the installation.

With VirtualBox installed, let’s move on to the next steps.

Preparing for Installation#

Before installing macOS Catalina, we need to make sure we have two files: the VirtualBox command file and the resolution file. These files are necessary for a successful installation.

  1. Open VirtualBox and click on ‘New’ to create a new virtual machine.
  2. Select ‘Expert Mode’ and enter ‘Catalina’ as the name of the machine.
  3. Keep the default settings for the machine folder and select ‘macOS X’ as the type.
  4. Make sure to select ‘64-bit’ as the version.
  5. Allocate at least 50% of your system’s RAM for the virtual machine.
  6. Select ‘Create a new virtual disk’ and click ‘Create’.
  7. Choose a location and set the disk capacity to at least 60GB.
  8. Select ‘Dynamically allocated’ as the storage type.
  9. Click ‘Create’ to create the virtual disk.

Next, we need to modify some settings in VirtualBox.

  1. Go to ‘Settings’ of the virtual machine.
  2. Click on the ‘System’ tab and go to the ‘Motherboard’ section.
  3. Select ‘PIIX3’ in the ‘Chipset’ dropdown menu.
  4. Go to the ‘Processors’ tab and allocate about 50% of the CPU power (e.g., 4 CPUs for a quad-core system).
  5. In the ‘Display’ tab, increase the video memory to the maximum (128MB).
  6. Choose ‘VMSVGA’ as the graphics controller.
  7. Under ‘Storage’, select the empty disk drive and choose the macOS Catalina ISO file you downloaded.
  8. Add the VirtualBox boot image file as a new hard drive.
  9. Under ‘USB’, select ‘USB 3.0’. Switch to ‘USB 2.0’ if you experience issues with the mouse.

After making these changes, you can shut down VirtualBox and move on to the next steps.

Running the VirtualBox Command Files#

  1. Open the Command Prompt as an administrator.
  2. Replace ‘VBox Name’ in the command files with the actual name of your VirtualBox machine.
  3. Execute the commands to change the directory and make the necessary adjustments.
  4. Once the commands have been executed, VirtualBox should be ready for the installation of macOS Catalina.

Installing macOS Catalina#

  1. Launch VirtualBox Manager and select the virtual machine.
  2. Click on ‘Start’ to begin the setup of the operating system.
  3. Select the language (English) and click ‘Continue’.
  4. Prepare the drive for the operating system by selecting ‘Disk Utility’ and clicking ‘Continue’.
  5. Choose the ‘vbox hard disk’ option and click on the ‘Erase’ icon.
  6. Enter a name for the drive and click ‘Erase’ to format it.
  7. After the formatting process is complete, click ‘Done’ and close the Disk Utility window.
  8. Select ‘macOS’ as the installation option and click ‘Continue’.
  9. Agree to the terms and conditions and select the drive you just formatted.
  10. Click ‘Install’ to start the installation process.
  11. Once the files have been copied, the system will automatically reboot.
  12. Select your country and click ‘Continue’.
  13. Set up the preferred language and input sources for the United States, or customize the settings if desired.
  14. Skip the transfer of files from an existing setup and click ‘Continue’.
  15. Sign in with your Apple ID if desired or click ‘Setup Later’.
  16. Agree to the terms and conditions and create an account for the computer.
  17. Click ‘Continue’ to proceed with the express setup.
  18. Customize settings if desired or click ‘Continue’ to leave them as default.
  19. Complete the setup process by agreeing to analytics and screen time settings.
  20. Choose the desired desktop interface (light mode or dark mode) and continue with the setup.

That’s it! You have successfully installed macOS Catalina on VirtualBox. You can now enjoy all the features that macOS Catalina has to offer on your Windows 10 PC.

Remember, if you encounter any issues during the installation, visit our forum at forum.geekgrower.com for help and solutions.

May the macOS Catalina experience on VirtualBox be productive and enjoyable for you!
