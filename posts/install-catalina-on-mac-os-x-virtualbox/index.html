Install macOS Catalina in VirtualBox (2021): A Comprehensive Guide

Are you a Windows user who wants to experience the features of macOS Catalina? Look no further! In this article, we will guide you through the step-by-step process of installing macOS Catalina on VirtualBox, allowing you to run the operating system on your Windows 10 PC. Let’s dive in.

System Requirements#

Before we begin, there are some system requirements that you need to meet in order to install macOS Catalina on VirtualBox. Here’s what you’ll need:

  1. Windows 10 PC
  2. At least 6GB of RAM (8GB is recommended)
  3. Minimum of 60GB of free hard disk space (80GB is preferable)
  4. Downloaded files, including VirtualBox and Extension Pack, Mac OS Catalina file, command file, and screen resolution file

To make it easier for you, we have provided download links for all the necessary files in the video description. These links will allow you to quickly access the files and proceed with the installation.

It’s important to note that not every installation goes smoothly. Depending on your hardware, such as AMD or Intel chips, you may encounter various problems. To provide assistance and solutions for these issues, we have created a forum on our website (forum.geekgrower.com) where you can ask questions, find solutions, and access additional files. Installation Steps#

Now, let’s get started with the installation process. If you haven’t already installed VirtualBox, follow these steps:

  1. Open your browser and visit the VirtualBox website.
  2. Download the appropriate version of VirtualBox for Windows.
  3. Additionally, download the Extension Pack.
  4. Once downloaded, run the installation for both VirtualBox and the Extension Pack.
  5. During the installation process, stick with the default settings and directories.
  6. After the installation is complete, open VirtualBox and click on “Preferences” in the VirtualBox Manager.
  7. In the preferences menu, navigate to the “Extensions” tab.
  8. Click on the “+” sign and add the downloaded Extension Pack.
  9. Install the Extension Pack by agreeing to the prompts.
  10. Once the installation is complete, you’re finished with the VirtualBox setup.

Preparing for Installation#

Before proceeding with the macOS Catalina installation, it’s essential to ensure that you have two files ready for the process: the VirtualBox command file and the resolution file. These files will allow you to successfully install Catalina on VirtualBox and adjust the screen resolution to your preference.

Creating a New Virtual Machine#

  1. Open VirtualBox and click on “New.”
  2. Select “Expert Mode.”
  3. Name your virtual machine as “Catalina.”
  4. Keep the machine folder as default.
  5. Select “macOS” as the type and “64-bit” as the version.
  6. Allocate roughly 50% of your system’s RAM to the virtual machine.
  7. Select “Create a new virtual disk” and click on “Create.”
  8. Leave the folder as default and set the disk capacity to 60GB.
  9. Choose “VMDK” and select “Dynamically allocated.”
  10. Click on “Create.”

Modifying Virtual Machine Settings#

  1. Open the settings for the newly created virtual machine.
  2. Click on the “System” tab and select “Motherboard.”
  3. Ensure that “piix3” is selected as the chipset.
  4. Go to the “Processors” tab and allocate 50% of your CPU power (e.g., 4 CPUs).
  5. Navigate to the “Display” tab and increase the video memory to 128MB.
  6. Choose “VBoxVGA” as the graphics controller.
  7. Under the “Storage” tab, select the empty disk drive and click on “Choose a disk file.”
  8. Locate and open the downloaded macOS Catalina ISO image.
  9. Add the VirtualBox boot image file by clicking on the “+” sign and selecting “Add Hard Drive.”
  10. Locate and open the VirtualBox boot image file.
  11. Under USB settings, select “USB 3.0” for better performance.
  12. Close VirtualBox to apply the settings.

Running the VirtualBox Command Files#

  1. Open the Command Prompt as an administrator.
  2. Replace the “vbox name” with the actual name of your VirtualBox (e.g., “Catalina”).
  3. Copy and paste the provided command lines into the Command Prompt.
  4. Execute the commands to manipulate the VirtualBox settings.
  5. Restart VirtualBox to enable the installation of macOS Catalina.

Installing macOS Catalina#

  1. Launch VirtualBox and select the virtual machine you created.
  2. Click on “Start” to begin the installation process.
  3. Choose your preferred language for the installation.
  4. Prepare the virtual hard drive by selecting “Disk Utility” and erasing the drive.
  5. Name the drive and click on “Erase.”
  6. Install macOS Catalina by selecting the previously created drive.
  7. Follow the on-screen instructions, agreeing to the terms and conditions.
  8. Customize the settings according to your preferences.
  9. Skip the option to transfer existing information to the virtual machine.
  10. Create an account for the operating system by entering your name and password.
  11. Complete the express setup, or customize settings as desired.
  12. Agree to the terms and conditions.
  13. Choose your desktop interface (light mode or dark mode) and finalize the setup.

Congratulations! You have successfully installed macOS Catalina on VirtualBox. You can now explore all the features and functionalities of macOS Catalina on your Windows 10 PC. Enjoy your virtual macOS experience!

In conclusion, this comprehensive guide has provided you with detailed steps on how to install macOS Catalina in VirtualBox on your Windows 10 PC. By following the outlined instructions, you can enjoy the full benefits of macOS Catalina without having to switch to a Mac computer. Happy computing!
